### 2.2 测试场景

**场景1: 正常流程**
1. 打开应用 → 按钮禁用 ✅
2. 截图/打开图片 → 按钮启用 ✅
3. 点击"识别" → 识别流程启动，按钮禁用 ✅
4. 识别完成 → 按钮启用，显示打码区域 ✅
5. 点击"识别"再次识别 → 重新执行识别 ✅

**场景2: 错误处理**
1. 网络异常时识别 → 显示错误信息 ✅
2. 错误后按钮恢复可用 → 可重试 ✅
