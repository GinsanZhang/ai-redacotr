## 1. 功能概述

### 1.1 需求背景
原始打码功能会将整个包含标签和敏感值的文字块全部模糊，导致用户无法识别被模糊的是什么类型的信息（如"姓名"、"电话"等）。本功能优化打码范围，只模糊敏感值部分，保留标签可见。

### 1.2 实现功能

✅ **核心功能**:
1. 智能识别标签和敏感值的分隔位置
2. 计算敏感值区域的精确边界框
3. 仅对敏感值部分应用打码效果
4. 保持标签部分清晰可见

✅ **支持场景**:
- 中文标签：姓名：张三
- 英文标签：Phone: 13800138000
- 多种分隔符：中文冒号、英文冒号（有无空格）

### 2.2 关键算法实现

#### 标签分离算法
```python
def split_label_value(text: str) -> tuple:
    """分离标签和敏感值"""
    for delimiter in LABEL_DELIMITERS:
        if delimiter in text:
            idx = text.index(delimiter)
            label = text[:idx + len(delimiter)]
            value = text[idx + len(delimiter):]
            return label, value
    return text, ""
```

#### 敏感区域计算
```python
def calculate_sensitive_bbox(bbox: tuple, label: str, text: str) -> tuple:
    """计算敏感值区域边界框"""
    x1, y1, x2, y2 = bbox
    total_width = x2 - x1
    label_ratio = len(label) / len(text) if text else 0
    label_width = int(total_width * label_ratio)
    sensitive_x1 = x1 + label_width
    return (sensitive_x1, y1, x2, y2)
```

## 6. 已知限制

1. **分隔符限制**: 目前仅支持预定义的3种分隔符（中文冒号、英文冒号）
2. **文本方向**: 假设文本为从左到右阅读
3. **等宽假设**: 计算基于字符等宽假设，对非等宽字体可能有偏差
