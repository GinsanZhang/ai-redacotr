## 2. 架构设计

### 2.1 流程对比

**当前流程（自动AI）**:
```
截图 → OCR(30-40s) → 规则(0.1s) → AI语义(30-40s) → 完成
总耗时: 60-80秒
```

**新流程 - 快速模式（默认）**:
```
截图 → OCR+敏感识别(30-40s) → 规则补充(0.1s) → 完成
总耗时: 30-40秒
```

**新流程 - 深度模式（可选）**:
```
截图 → OCR(30-40s) → 规则(0.1s) → AI语义(30-40s) → 完成
总耗时: 60-80秒
```

#### 3.1.3 ui.py - 修改 ProcessWorker

```python
class ProcessWorker(QThread):
    def __init__(self, image_cv: np.ndarray):
        super().__init__()
        self.image_cv = image_cv
        self.signals = ProcessSignals()
    
    def run(self):
        try:
            # 根据配置选择流程
            if CONFIG.get("deep_ai_enabled", False):
                self._run_deep_mode()
            else:
                self._run_fast_mode()
        except Exception as e:
            self.signals.error.emit(str(e))
    
    def _run_fast_mode(self):
        """快速模式：OCR同时识别敏感信息 + 规则补充"""
        timings = {}
        total_start = time.time()
        
        # Phase 1: OCR + 敏感信息识别 (30-40s)
        self.signals.progress_value.emit(5)
        self.signals.progress.emit("云端视觉识别中...")
        s1 = time.time()
        
        result = detect_by_cloud_vision(
            self.image_cv, 
            self.signals.progress.emit,
            include_sensitive=True  # 同时识别敏感信息
        )
        
        ocr_blocks = result["ocr_blocks"]
        ai_hits = result["sensitive_hits"]  # OCR识别的敏感信息
        
        timings["云端视觉"] = time.time() - s1
        self.signals.progress_value.emit(80)
        
        # Phase 2: 规则匹配补充 (0.1s)
        self.signals.progress.emit("规则匹配中...")
        s2 = time.time()
        rule_hits = detect_by_rules(ocr_blocks)
        
        # 合并结果，去重（规则匹配到的不再添加）
        rule_texts = {h["text"] for h in rule_hits}
        combined_hits = rule_hits + [
            h for h in ai_hits 
            if h["text"] not in rule_texts
        ]
        
        timings["规则匹配"] = time.time() - s2
        timings["总耗时"] = time.time() - total_start
        
        self.signals.progress_value.emit(100)
        self.signals.progress.emit("处理完成")
        self.signals.finished.emit(ocr_blocks, combined_hits, timings)
    
    def _run_deep_mode(self):
        """深度模式：原流程（OCR + 规则 + AI语义）"""
        timings = {}
        total_start = time.time()
        
        # Phase 1: OCR (30-40s)
        self.signals.progress_value.emit(5)
        self.signals.progress.emit("云端视觉识别中...")
        s1 = time.time()
        ocr = detect_by_cloud_vision(
            self.image_cv, 
            self.signals.progress.emit,
            include_sensitive=False  # 仅OCR
        )
        timings["云端视觉"] = time.time() - s1
        self.signals.progress_value.emit(40)
        
        # Phase 2: 规则匹配 (0.1s)
        self.signals.progress.emit("规则匹配中...")
        s2 = time.time()
        rules = detect_by_rules(ocr)
        timings["规则匹配"] = time.time() - s2
        self.signals.progress_value.emit(50)
        
        # Phase 3: AI语义分析 (30-40s)
        self.signals.progress.emit("AI分析中...")
        s3 = time.time()
        rule_texts = {h["text"] for h in rules}
        remaining = [b for b in ocr if b["text"] not in rule_texts]
        ai = detect_by_ai(remaining, self.signals.progress.emit)
        timings["AI分析"] = time.time() - s3
        timings["总耗时"] = time.time() - total_start
        
        self.signals.progress_value.emit(100)
        self.signals.progress.emit("处理完成")
        self.signals.finished.emit(ocr, rules + ai, timings)
```
