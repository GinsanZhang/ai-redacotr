## 版本特性

### 核心功能
- ✅ 区域截图、全屏截图、本地图片打开
- ✅ 云端视觉识别（OCR）
- ✅ 规则匹配识别（8种标准格式）
- ✅ AI 语义识别（姓名、地址等）
- ✅ 手动添加/删除打码区域
- ✅ 三种打码样式（高斯模糊、像素马赛克、纯黑遮罩）
- ✅ 图片保存与复制到剪贴板
- ✅ 全局快捷键支持

### 技术特性
- ✅ PyQt6 框架（支持高 DPI）
- ✅ 多线程处理（不阻塞 UI）
- ✅ 流式 API 响应（实时进度）
- ✅ 优雅降级（AI 失败不影响使用）
- ✅ 坐标偏移修复（手动打码精准）

## 使用说明

### 快速开始
```bash
# 安装依赖
pip install -r requirements.txt

# 配置 API Key
set LLM_API_KEY=your_api_key

# 运行
python -m src.redactor
```

### 快捷键
- `Ctrl+Shift+S` - 区域截图
- `Ctrl+Z` - 撤销
- `Ctrl+S` - 保存图片
- `Ctrl+C` - 复制到剪贴板
