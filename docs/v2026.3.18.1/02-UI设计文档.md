# UI设计文档
## 截图打码助手 - 版本 v2026.3.18.1

---

## 1. 设计概述

### 1.1 设计目标
- 打造专业、高效的隐私打码工具界面
- 采用深色主题降低视觉疲劳
- 强调核心操作区域，弱化次要元素
- 保持界面简洁，减少认知负担

### 1.2 设计语言
- **风格**: 现代扁平化 + 微拟物
- **布局**: 三栏式布局（工具栏 | 主画布 | 侧边栏）
- **动效**: 即时反馈，无过渡动画（追求极速响应）

## 3. 布局设计

### 3.1 整体布局

```
┌──────────────────────────────────────────────────────────────┐ ← 52px
│ ToolBar (Fixed Height: 52px)                                  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                    Main Content Area                          │
│                                                              │
│   ┌────────────────────────────────────┐  ┌──────────────┐  │
│   │                                    │  │              │  │
│   │      Canvas (ScrollArea)           │  │  Right Panel │  │
│   │      (Adaptive)                    │  │  (220px)     │  │
│   │                                    │  │              │  │
│   │                                    │  │              │  │
│   └────────────────────────────────────┘  └──────────────┘  │
│                                                              │
├──────────────────────────────────────────────────────────────┤ ← 24px
│ StatusBar (Fixed Height: 24px)                              │
└──────────────────────────────────────────────────────────────┘
         ↑
    Min Width: 960px
    Min Height: 680px
```

### 3.2 窗口尺寸规范

| 参数 | 值 | 说明 |
|------|-----|------|
| 最小宽度 | 960px | 保证功能完整显示 |
| 最小高度 | 680px | 保证画布可操作 |
| 默认宽度 | 1100px | 初始窗口大小 |
| 默认高度 | 760px | 初始窗口大小 |
| 内边距 | 12px | 内容区域边距 |
| 间距 | 12px | 组件间间距 |

### 3.3 响应式断点

- **小屏 (< 1200px)**: 侧边栏折叠为图标模式（预留）
- **中屏 (1200-1600px)**: 标准三栏布局
- **大屏 (> 1600px)**: 侧边栏可加宽至 260px（预留）

## 4. 组件设计

### 4.1 工具栏 (Toolbar)

#### 结构
```
┌────────────────────────────────────────────────────────────────┐
│ 🔒 截图打码    │ [截图组] │ [模式组] │ [编辑组] │ [导出组]      │
└────────────────────────────────────────────────────────────────┘
   16px          间距 8px    间距 16px  间距 16px   间距 16px
```

#### 按钮规范

**主按钮 (btnPrimary)**
```css
background: #3b5bdb;
color: white;
border: none;
border-radius: 8px;
padding: 7px 14px;
font-weight: 600;
:hover { background: #2f4cbf; }
```

**次按钮 (btnSecondary)**
```css
background: #1e2030;
color: #94a3b8;
border: 1px solid #2d3148;
border-radius: 8px;
padding: 7px 14px;
:hover { background: #252840; color: #e2e8f0; }
```

**模式按钮 (btnMode)**
```css
background: transparent;
color: #64748b;
border: 1px solid #2d3148;
:checked {
  background: #1e3a5f;
  color: #60a5fa;
  border-color: #3b82f6;
}
```

**成功按钮 (btnSuccess)**
```css
background: #064e3b;
color: #34d399;
border: 1px solid #065f46;
:disabled { background: #1a2020; color: #2d4a3a; }
```

**危险按钮 (btnDanger)**
```css
background: transparent;
color: #64748b;
:hover { background: #2d1515; color: #f87171; border-color: #f87171; }
```

### 4.2 画布区域 (Canvas)

#### 基础样式
- 背景色: `#0d0d12`
- 最小尺寸: 400x300
- 对齐方式: 居中 (AlignCenter)
- 鼠标样式: 十字准星 (CrossCursor)

#### 打码区域视觉

**规则匹配区域**
```
边框: 2px solid rgba(255, 80, 80, 0.7)
填充: rgba(255, 80, 80, 0.12)
标签: 白色文字 + 半透明底色
```

**AI识别区域**
```
边框: 2px solid rgba(255, 160, 0, 0.7)
填充: rgba(255, 160, 0, 0.12)
标签: 白色文字 + 半透明底色
```

**手动添加区域**
```
边框: 2px solid rgba(100, 200, 255, 0.7)
填充: rgba(100, 200, 255, 0.12)
标签: 白色文字 + "手动"
```

**悬停高亮**
```
边框: 2px solid rgba(255, 50, 50, 0.86)
填充: rgba(255, 50, 50, 0.18)
```

**拖拽绘制预览**
```
边框: 2px dashed rgba(100, 200, 255, 1)
填充: rgba(100, 200, 255, 0.16)
```

### 4.4 截图选择器 (ScreenshotSelector)

#### 视觉效果
- 全屏遮罩: `rgba(0, 0, 0, 0.4)`
- 选择区域: 透明挖空 + 蓝色边框
- 尺寸提示: 白色文字，显示宽高
- 光标: 十字准星

#### 交互
- 按下左键开始选择
- 拖拽时实时显示选择框
- 释放左键完成截图
- 按 ESC 取消

## 7. 交互动效

### 7.1 按钮反馈
- **悬停**: 背景色变化 (瞬时)
- **按下**: 轻微缩放 (预留)
- **禁用**: 透明度 50%

### 7.2 区域反馈
- **悬停**: 边框加粗 + 颜色加深
- **选中**: 显示标签
- **绘制中**: 实线变虚线

### 7.3 进度反馈
- 进度条: 平滑填充动画
- 阶段文字: 即时更新
- 统计信息: 处理完成后更新

## 8. 响应式适配

### 8.1 高 DPI 支持
- 使用 Qt 的高 DPI 缩放机制
- 图片坐标正确处理 DPR (Device Pixel Ratio)
- 界面元素自动缩放

### 8.2 小屏幕适配
- 画布最小尺寸保证可操作性
- 侧边栏内容可滚动
- 工具栏按钮保持可点击
