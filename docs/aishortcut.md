## 功能清单

**自动识别**：手机号、身份证、银行卡、邮箱、IP、车牌、社会信用代码（正则，无需AI）+ 姓名/地址等非标准信息（AI，可选）

**手动补充**：左键拖拽任意区域添加打码，切换删除模式点击移除误判

**打码样式**：高斯模糊 / 像素马赛克 / 纯黑遮罩，强度可调

**导出**：保存PNG/JPG，或直接复制到剪贴板

## 第一步先装依赖

```bash
pip install PyQt6 paddleocr paddlepaddle pillow opencv-python requests keyboard
```

PaddleOCR 第一次运行会下载模型（~500MB），之后就快了。不配置 API Key 也能直接用，只是姓名/地址等非标准信息需要手动补充打码。遇到问题随时贴报错给我。
