# 领域词汇表 (Domain Vocabulary)

本文档定义了截图打码工具项目中使用的统一术语规范，确保代码、文档和界面文案的一致性。

---

## 核心概念术语

### VLM (Vision Language Model) - 视觉语言模型
**定义**: 能够同时处理图像和文本的多模态AI模型，用于OCR文字识别和视觉理解。

**使用场景**:
- 代码配置: `vlm_api_key`, `vlm_model`, `vlm_timeout`
- 环境变量: `VLM_API_KEY`, `VLM_MODEL`
- 界面文案: "VLM识别", "VLM模型"
- 函数名: `detect_by_vlm()`

**禁止使用的旧术语**:
- ❌ 云端视觉 (cloud_vision)
- ❌ 多模态识别
- ❌ cloud_vision_*

---

### LLM (Large Language Model) - 大语言模型
**定义**: 用于文本理解和语义分析的大型语言模型，用于识别姓名、地址等需要语义理解的敏感信息。

**使用场景**:
- 代码配置: `llm_api_key`, `llm_model`, `llm_timeout`
- 环境变量: `LLM_API_KEY`, `LLM_MODEL`
- 界面文案: "LLM", "LLM语义分析"
- 函数名: `detect_by_ai()` (保持现有)

**禁止使用的旧术语**:
- ❌ AI大模型
- ❌ AI 大模型
- ❌ 大模型语义分析 (应改为 LLM语义分析)

---

## 功能模块术语

### OCR (Optical Character Recognition) - 光学字符识别
**定义**: 从图片中提取文字的技术，在本项目中由VLM实现。

**使用场景**:
- 代码注释: "OCR识别"
- 文档描述: "OCR文字识别"
- 进度提示: "VLM识别中..." (具体实现)

---

### 敏感信息识别 (Sensitive Information Detection)
**定义**: 识别图片中的个人隐私信息，如手机号、身份证号、姓名等。

**使用场景**:
- 界面: "敏感信息识别"
- 文档: "隐私信息检测"
- 配置: `include_sensitive` 参数

---

## 配置项命名规范

### 环境变量命名
**格式**: `模块_配置项`，全大写，下划线分隔

**示例**:
```bash
# VLM模块
VLM_API_URL=https://api.siliconflow.cn/v1/chat/completions
VLM_API_KEY=your_key_here
VLM_MODEL=Qwen/Qwen3-VL-32B-Instruct
VLM_TIMEOUT=180
VLM_RETRIES=1

# LLM模块
LLM_API_URL=https://api.siliconflow.cn/v1/chat/completions
LLM_API_KEY=your_key_here
LLM_MODEL=Pro/moonshotai/Kimi-K2.5
LLM_TIMEOUT=180
LLM_RETRIES=2
```

**禁止使用的旧格式**:
- ❌ `CLOUD_VISION_API_URL`
- ❌ `CLOUD_VISION_API_KEY`

---

### 配置字典键名
**格式**: `模块_配置项`，全小写，下划线分隔

**示例**:
```python
CONFIG = {
    "vlm_api_url": "...",

    "vlm_model": "...",
    "llm_api_url": "...",
    "llm_api_key": "...",

```
